# Manual de Usuario - Vecino Alerta

## Introducción
Vecino Alerta es un sistema integral de seguridad comunitaria que permite a los vecinos reportar emergencias y actividades sospechosas, activando sirenas disuasivas y notificando a la comunidad.

## Para Vecinos

### 1. Instalación y Registro
1. Descargue la aplicación "Vecino Alerta" (Android/iOS).
2. Abra la app y regístrese con su número de teléfono.
3. Ingrese el código de verificación SMS.
4. Espere a que el Comité de su barrio apruebe su solicitud.

### 2. Botón de Pánico (Emergencia)
- **Activar:** En la pantalla principal, mantenga presionado el **BOTÓN ROJO** por 2 segundos.
- **Qué sucede:**
  - Se activa la sirena del barrio.
  - Se envía una notificación a todos los vecinos.
  - Su ubicación se comparte en tiempo real.
- **Cancelar:** Si fue un error, presione "Cancelar" e ingrese su PIN o confirme la cancelación.

### 3. Reporte Silencioso
- Use el botón "Reporte Silencioso" para informar actividad sospechosa sin activar la sirena.
- Puede adjuntar detalles y tipo de incidente.

## Para el Comité

### 1. Panel de Administración
Acceda al panel web en: `https://[SU-PROYECTO].web.app`

### 2. Gestión de Usuarios
- Apruebe o rechace nuevos vecinos en la sección "Usuarios".
- Asigne roles (Vecino, Comité, Guardian).

### 3. Mapa de Calor
- Visualice las zonas con más incidentes en el "Dashboard".
- Use esta información para mejorar la seguridad preventiva.

## Soporte
Para ayuda técnica, contacte al administrador del sistema o al Comité de Seguridad.
